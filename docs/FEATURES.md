# Leafly Features

## Authentication
- Email/Password login and registration
- Google Sign-In
- Forgot password (email reset)
- Auto-login if already logged in
- Delete account

## Plant Management
- Add plants with photos (free, no Storage needed)
- Edit plant details
- Delete plant with confirmation
- Search plants by name or species
- View plant details

## Care Tasks
- Add care tasks per plant
- Mark tasks as done/undone
- Delete tasks
- Due date tracking

## Growth Logs
- Add growth notes per plant
- View logs sorted by date
- Delete logs

## AI Plant Care
- Chat with Groq Llama 3.3 AI
- Get personalized plant care tips
- Save tips to Firestore
- View saved tips anytime
- AI bot on Dashboard

## Reminders
- Watering reminders via WorkManager
- Periodic notifications based on watering frequency
- Test reminder (5 seconds)

## Profile
- Edit name, contact, address
- Profile picture (Base64, free)
- Dark/Light theme toggle
- Logout
- Delete account

## Dashboard
- Real-time plant stats
- Today's tasks
- My plants carousel with images
- AI bot widget
- Quick actions (Export, Tips, Reminders)

## Export
- Export all plants as CSV
- Share via any app
